## 🗺️ About This Project

A text-based **choose your own adventure** game built in Python. You play as an explorer trying to find hidden treasure on a mysterious island. Every choice you make leads to a different outcome — but only one path leads to victory!

---

## 🧠 What I Learned

- `if`, `elif`, and `else` statements
- Nested `if` statements
- Comparison operators (`==`, `!=`, `>`, `<`)
- Logical operators (`and`, `or`, `not`)
- Taking user input with `input()`
- String methods like `.lower()`

---

## ▶️ How to Run

```bash
python main.py
```

Make sure you have Python 3 installed. No extra libraries needed!

---

## 🎮 How to Play

You will be presented with choices at each stage. Type your answer and press Enter.

There are **3 stages**:
1. A crossroads — go left or right
2. A lake — wait or swim
3. A house with 3 doors — red, yellow, or blue

Only **one combination** leads to the treasure. Good luck! 🍀
